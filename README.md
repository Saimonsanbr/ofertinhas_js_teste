# ofertinhas_js_teste
Esse é um teste pessoal de um frontend que bebe de uma api pública (frontend) de um site de ofertas. O objetivo é conseguir as ofertas, os cupons, os links e informações dos produtos, usando requisições js apenas, com html, css e js básico. Não há alterações na api deles, nos links, nem nada além de requisições normais, simulando o frontend deles.
